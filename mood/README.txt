MOOD FOLDER
===========

Place your image files in this folder to display them in the mood board.

Supported formats: .jpg, .jpeg, .png, .gif, .webp, .svg

After adding images:
1. Open index.html
2. Update the imageFiles array with your image filenames
3. Save and refresh the page

Example:
const imageFiles = [
    'photo1.jpg',
    'photo2.png',
    'design3.jpg',
];
